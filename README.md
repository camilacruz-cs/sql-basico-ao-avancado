# SQL para Análise de Dados

Este repositório contém minha jornada de aprendizado em SQL com foco em análise de dados. Os conteúdos foram desenvolvidos a partir do curso *SQL para Análise de Dados*, ministrado por Midori Toyota.

## Objetivo

Consolidar conhecimentos em SQL e evoluir na construção de análises orientadas a dados, com foco em aplicações reais de negócio.

## O que você vai encontrar

- Queries organizadas por nível de complexidade
- Exemplos práticos aplicados a cenários de negócio
- Uso de boas práticas como CTEs, joins e agregações

## Principais aprendizados

- Como estruturar queries para análise exploratória
- Boas práticas de organização de código SQL
- Uso de SQL para responder perguntas de negócio

## Estrutura do repositório

- `01_fundamentos` → comandos básicos e operadores  
- `02_agregacoes` → funções agregadoras e GROUP BY  
- `03_joins` → integração de tabelas  
- `04_unions` → combinação de datasets  
- `05_subqueries` → subconsultas  
- `06_transformacao_dados` → limpeza e tratamento de dados  
- `07_manipulacao_tabelas` → criação e alteração de tabelas  
- `08_projetos` → aplicações práticas


## Projetos

- Análise de funil de vendas
- Segmentação de clientes

## Tecnologias

- SQL (PostgreSQL)
